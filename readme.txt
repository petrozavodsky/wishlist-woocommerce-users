WooCommerce adds wishlist button to product and helps customers to save their favorite products in a wishlist page.

== Description ==

Wishlist WooCommerce is a simple and light weight plugin which adds wishlist button to product archive page and also on single product page.
Users can manage their all favorites products at one place.

### Features And Options:
* Customizable Wishlist icon.
* Fancy Dialog box when product is added to wishlist.
* Undo button when product is removed from wishlist.
* Choose your own position for wishlist button.
* Shortcode to change your wishlist page.
* Option to include/exclude on single product page.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Click on Wishlist menu on the dashboard.

== Frequently Asked Questions ==

= How to setup? =

Everything is already setup.Just activate the plugin & you are done.

* Shortcode for Products Wishlist?

`[vp_wishlist]` (By default , Wishlist WooCommerce creates Wishlist page.)
